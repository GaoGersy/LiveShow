# LiveShow
使用聚合数据接口制作的一款集成了搞笑图片，笑话大全，微信精选，天气等的在线App

[![Hex.pm](https://img.shields.io/hexpm/l/plug.svg)]()

[![platform](https://img.shields.io/badge/platform-Android-yellow.svg)](https://www.android.com)
```技术一览：```
- 1.使用MVP架构
- 2.使用RxJava处理异步任务
- 3.采用Fresco进行Gif图片的展示
- 4.采用RecylerView展示数据
